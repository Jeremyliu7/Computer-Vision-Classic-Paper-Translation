## 计算机视觉论文翻译计划   
    计算机视觉论文翻译计划是一个翻译优质计算机视觉技术文章的项目，文章来源为CVPR,ICCV,ECCV上的优质论文。内容覆盖目标检测，实例分割，移动端图像算法等领域，以及各大型优质官方文档及手册，读者为想从事计算机视觉研究的兴趣爱好者和热爱人工智能的新锐开发者。
